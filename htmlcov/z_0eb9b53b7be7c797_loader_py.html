67 """Scan ``path`` for ``*.md`` files and parse each into an :class:`AgentDefinition`.

70 Dot-prefixed files and ``*.template.md`` reference templates are skipped. 

71 Any agent whose frontmatter omits ``tools:`` is normalized to receive

73

80 raise FileNotFoundError(f"agents directory does not exist: {path}")
